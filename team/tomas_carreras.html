I'm Tomás Carreras, I was a programmer for this project. I changed a lot between scrum teams, adding new features in different places. In this project I learned a lot of new skills, like in every project year, like using OpenGL to render 2D and 3D at the same time. This has been the first time I have worked on a team with more than 3 programmers and I have had a lot of things to learn because of it, since we had to work on a lot of different branches to avoid problems and we also had to maintain active communication and cooperation. Having touched so many areas, I have really come to know how most of the areas of the code works and how my teammates work which has been an extremely good experience.
